> 
> Responsive
> 
> Designed by me ( logo, setapp nyan cat, etc.)  ;-)


## Installation

1. Download repository
2. Get into project directory -> `npm install`
3. `npm start`

## Based on

1. node version v10.14.1
2. npm version -> 6.4.1
3. react version -> 16.12.0

## Features

1. Album search
2. Album "more info" modal
3. ScrollTop component ;-)
4. Nyan cat loader
